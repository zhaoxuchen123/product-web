## 初始化示例

```c
#include <nic_driver.h>

static nic_dev_t g_nic;

void net_init(void)
{
    nic_cfg_t cfg = {
        .type     = NIC_TYPE_I40E,
        .pcie_bus = 0x05,
        .pcie_dev = 0x00,
        .pcie_fn  = 0x00,
        .irq      = 24,
        .mode     = NIC_MODE_ISR,
    };
    nic_open(&g_nic, &cfg);
}
```

## 固件要求

i40e 系列需要网卡本身的 NVM 固件版本 ≥ 6.01。可通过以下方式查询：

```c
nic_hw_info_t info;
nic_ioctl(&g_nic, NIC_IOCTL_GET_HW_INFO, &info);
/* info.fw_ver: 固件版本字符串 */
```

若固件版本过低，驱动初始化会返回 `-ENOTSUP`，需通过 Intel 官方工具更新 NVM。

## 性能指标

| 测试场景 | 型号 | 吞吐量 | CPU 占用 |
|----------|------|--------|----------|
| UDP 单流发送 | X710 | 9.8 Gbps | 4.1% |
| UDP 单流接收 | X710 | 9.8 Gbps | 4.5% |
| TCP 双向   | X710 | 9.6 Gbps | 8.2% |
| RSS 8队列  | X710 | 9.8 Gbps | 2.1% |

> 测试环境：SylixOS x86，X710-DA2，MSI-X，RSS 8 队列

## 多队列 RSS 配置

X710 支持最多 64 个 RSS 队列，大流量场景下充分利用多核：

```c
nic_rss_cfg_t rss = {
    .rx_queues = 8,
    .tx_queues = 8,
    .hash_type = NIC_RSS_HASH_TCP4 | NIC_RSS_HASH_TCP6,
};
nic_ioctl(&g_nic, NIC_IOCTL_SET_RSS, &rss);
```
